## Como rodar localmente
- abrir `index.html` diretamente

## Pontos de entrada / arquivos importantes
- `index.html` — página principal (HTML)
- `css/style.css` — estilos
- `js/main.js` — scripts principais
- `docs/DOCUMENTATION.md` — documentação do projeto
- `assets/img/` — imagens e avatar

## Ambiente recomendado
- OS: Windows
- Shell: PowerShell
- Navegadores alvo: Chrome, Edge, Firefox (versões modernas)

## Convenções
- Commits: Conventional Commits (ex.: feat(...): descrição)
- Links externos: usar `target="_blank" rel="noopener noreferrer"`
- Acessibilidade: manter `aria-*` e uso semântico de headings

## Permissões de edição
- Arquivos que o assistente pode editar sem pedir: `css/`, `js/`, `docs/`, `index.html` (conteúdo textual)
- Arquivos que NÃO devem ser alterados sem aprovação: `config/`, `secrets/`, `db/migrations`

## Dados sensíveis
- Não incluir chaves/segredos neste repositório. Use `.env` ou fluxo seguro para credenciais.
